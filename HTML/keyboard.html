3D Map Editor Ver 2.24

KEYBOARD SHORTCUTS:

To help you editing fastly , shortcuts were created that are equivault to function defined by icons . They can be use in FLY-MODE :

F1 = Load a map from disk
F2 = Clear the current map
F3 = Save the current map to disk
F4 = Save camera state
F5 = Load a new texture from disk
F6 = Kill a texture from map
F7 = Load a 3D Direct X Object (XOF)
F8= Kill a 3D Direct X Object ( Not yet )
F9 = Load a new 3D Sound from disk
F10 = Kill a 3D sound from map ( Not yet )
1 = Add a new light to map
2 = Add a 3D Direct X Object to map
3 = Add a new 3D Sound to map
4 = Add a sphere to map
5 = Add a cube to map
6 = Add a box to map
7 = Add a cylinder to map
8 = Add a cone to map
9 = Add a plain to map
c = Copy object
m = move object
d = scale object
r = rotate object
a = object attribute
o = optimize objects position order
g = current object ghost mode
s = mode select object
p = preview mode / normal mode
PageUP = Scroll texture preview Up
PageDOWN = Scroll texture preview Down
InserKey = Apply next texture to object
Suppr = Apply previous texture to object
ESC / F12 = Quit 3D MapEditor
