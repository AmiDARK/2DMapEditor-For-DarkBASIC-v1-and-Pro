3D Map Editor Ver 2.31

( Html documentation )

3DMapEditor un a program created with DarkBasic . It's a tool to help in the creation of 3D worlds for DarkBasic's games.For the moment , 3DMapEditor can create primitives objects sich like Sphere , cube , boxex , cylinders , cones and plains. You can also adding light sources ( with a maximum of 256 lights ) . You can modify the state of the ambient light ( colour , amplitude and direction ).You can also adding directly DirectX 3DObjects. I Will adding others features like Matrix , 3D Sounds , and others stuffs...

I hope you will enjoy it ...
