3D Map Editor Ver 2.24

RACCOURCIS CLAVIER :

Pour simplifier la tache plusieurs raccourcis clavier ont été crées . Ils sont effectifs dans le mode FLY :

F1 = Charger une carte
F2 = Effacer la carte courante
F3 = Sauvegarder la carte vers le disque
F4 = Sauver les attributs de la camera
F5 = Ajouter une texture du disque
F6 = Détruire une texture de la carte
F7 = Charger un object 3D DirectX (XOF)
F8 = Détruire un object 3D DirectX de la carte ( non inplémenté )
F9 = Charger un son 3D
F10 = Détruire un son 3D ( non inplémenté )
1 = Ajouter une lumière
2 = Ajouter un objet 3D Direct X
3 = Ajouter un son 3D
4 = Ajouter une sphère
5 = Ajouter un cube
6 = Ajouter une boite
7 = Ajouter un cylindre
8 = Ajouter un cone
9 = Ajouter un plan
c = Copier l'objet courant
m = déplacer l'objet courant
d = redimensionner l'objet courant
r = appliquer des rotations à l'objet courant
a = attributs de l'objet en cours
o = optimizer les priorités des objets par distances
g = mode de transparence de l'objet courant
s = mode de sélection d'objet
p = passage de preview à mode normal
PageUP = Faire défiles les prévisualisation des textures vers le haut
PageDOWN = Faire défiles les prévisualisation des textures vers le bas
InserKey = Appliquer la texture suivante à l'objet courant
Suppr = Appliquer la texture précédente à l'objet courant
ESC / F12 = Quitter 3D MapEditor
