3D Map Editor Ver 2.25+

PRIMITIVES :

The 7 followinfg icons allow user to adding objects :

---- ------------------------

Sphere__Cube___Box__Cylinder__Cone___Plain___Light___Sound

The 3 following icons modify directly the current object :

---- ---- ----

Move__Rotate___Scale___Copy

MOVE , ROTATE and SCALING mode uses Left,Right key to modify on X axes , Up,Down key for Y axes and PageUP/PageDown for Z axes.and finally Inser/Supprim key to modify steps.

With that icon you can change current object , go back to an ancient object and modify it again. You can then use PageUP/PageDown to select the object you want to modify.

____

With these 4 icons you have full access to textures. The 2 first modify directly the current object texture by appling the previous or the next texture . The 2 last allow the user to change preview of texture ( on the right of the screen ) to display all loaded textures.

With tha icon you can change the object mode , normal , transparency and dark transparency..

And Finally , with that icon you can scale texture that are applied to the current object..

That icon is a special icon.When you select that option , 3DMapedit will remap the order of the objects presents in the map regarding to the distance on each one between each other . Example : If objects 4 and 156 are near then object 156 will become object 5 .( 3DMapEdit ver 2.20+ )

That object allow the user to modify attributes of the current object . It's equal to Dark basic command : "SET OBJECT x,a,b,c,d,e,f,g" .

With that icon you'll be able to kill any 3D object from the map.
