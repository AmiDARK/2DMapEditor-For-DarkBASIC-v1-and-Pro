3D Map Editor Ver 2.00+

DISK icons actives in Ver 2.11 :

" LOAD MAP " : Load a previously saved map from disk .

" SAVE MAP " : Save the current map to disk .

" CLEAR MAP " : Clear entirely the current map

" QUIT " : Quit editor.

" LOAD TEXTURE " :Load a new texture and adding it to texture list.
