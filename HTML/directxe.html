3D Map Editor Ver 2.25+

Direct X 3D Objects :

The following icons affects Direct X 3D Objects :

That icon add a new DirectX 3D Object ( .x ) to list of 3DObjects in map .

That icon add a DirectX 3D Object in map.

With that icon you'll be able to remove any Direct X Object from the list ( and all child 3D object that were created with the deleted Direct X Object ) ( ver 2.25 )

These are fully functionnal with Direct Object that were added to the map.

BEWARE : Version 2.15 do not save DirectX 3D Object , New version save them.
