3D Map Editor Ver 2.30


EXTRAS #2 :Matrixes

Add Matrix :
add a new matrix to the map.

Move Matrix :
allow user to move a matrix everywhere in the map.

Edit Heights :
Allow user to change the height of any point of the current matrix..
There are keyboards shortcuts for FAST changes :
key 'X' make all the points on the same X coordinate to have the same height than the current point
key 'Z' make all the points with the same Z coordinate to have the same height than the current point
key 'A' make all matrix point to have the same height than the current point

Edit Tiles :
Allow user to change the texture of each tile separately.
there are keyboards shortcuts for FAST changes :
key 'X' make all the points on the same X coordinate to use the same texture than the current point
key 'Z' make all the points with the same Z coordinate to use the same texture than the current point
key 'A' make all matrix point to use the same texture than the current point

Sélect Matrix :
Allow user to choose a specified existing matrix to modify it.

Matrix Properties :
Open the matrix control panel to change parameters.

Kill Matrix :
delete the current matrix from the map

Matrix Rnd(y) :
Can define randomly all the matrix point heights using minima-maxima and steps parameters.

Matrix Terrain :
unavailable in 2.30
Allow user to use a .bmp or .jpg image file to create matrix heights like terrain does . . .

Make Noise :
unavailable in 2.30
can make noise effect on matrix to make it a bit realistic .
