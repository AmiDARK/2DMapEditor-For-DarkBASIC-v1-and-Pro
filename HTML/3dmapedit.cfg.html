3D Map Editor Ver 2.30
Comment configurer 3DMapEditor

[REGISTERED USER]
Cette variable permet d'entrer le nom d'utilisateur enregistré pour activer le mode COMPLET

[REGISTERED ID]
Cette variable permet d'entrer le code correspondant au nom d'utilisateur enregistré pour activer le mode COMPLET

[LANGUAGE]
Cette variable permet de choisir quel langage 3dmapeditor utilisera. Les valeurs valides sont :
FRANCAIS , ENGLISH

[RESOLUTION]
Cette variable permet de choisir quelle résolution graphique 3DMapEditor Utilisera.
Les valeurs valides sont les suivantes : 640x480 , 800x600 , 1024x768 , 1280x1024

[FOG]
Cette variable nécessite 4 lignes de paramètres.Dans l'ordre ces 4 lignes contiendront :
Les composantes de couleur Rouge , vert , bleu ainsi que la distance de l'effet de brouillard.

[BACKGROUND]
Cette variable nécessite 3 lignes de paramètres.Dans l'ordre ces 3 lignes contiendront :
Les composantes de couleur Rouge , vert et bleu de la couleur de fond d'écran.

[TEXTUREMEM]
Cette variable permet de définir quel type de mémoire 3DMapEditor utilisera pour stoquer les textures.
VIDEORAM utilisera la mémoire de la carte graphique ( rapide )
RAMONLY utilisera la mémoire système ( plus lent mais pratique pour certaines cartes graphiques. )

[TEXTURETYPE]
Cette variable définit si 3DMapEditor utilisera la compression de texture matérielle ( TRUE ) si la carte vidéo le permet.
Ou le mode standart (FALSE ) .
Attention le mode TRUE ne fonctionns que si [TEXTUREMEM] est en mode VIDEORAM

[MATRIXSIZE]
Définit combien de cases la matrice de calage de 3DMapEditor sera composée.

[MAPPATH]
Définit le chemin d'accès relatif aux fichiers maps (.map) de 3DMapEditor.

[TEXTUREPATH]
Définit le chemin d'accès relatif aux fichiers de textures (.bmp , .jpg) pour les maps de 3DMapEditor.

[3DOBJECTPATH]
Définit le chemin d'accès relatifs aux fichiers d'objets 3D (.x , .3ds) pour les maps de 3DMapEditor.

[3DSOUNDPATH]
Définit le chemin d'accès relatif aux fichiers sons 3D (.wav , .mp3) pour les maps de 3DMapEditor.

[DEBUGMODE]
Active le mode DEBUG de 3DMapEditor pour les vérifications de débogage ( à ne pas utiliser )
