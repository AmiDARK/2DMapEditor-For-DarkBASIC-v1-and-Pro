3D Map Editor Ver 2.30
How to configure 3DMapEditor . . .

[REGISTERED USER]
That variable allow user to enter registered name to activate FULL-MODE

[REGISTERED ID]
That variable allow user to enter registered code to activate FULL-MODE

[LANGUAGE]
That variable allow to choose which langage 3DMapEditor will use. Valid values are :
FRANCAIS , ENGLISH

[RESOLUTION]
Allow to change the used graphic resolution for 3DMapEditor screen.
You can choose between these values : 640x480 , 800x600 , 1024x768 , 1280x1024

[FOG]
That variable need 4 parameters lines.These lines must contain in order :
Red , Green , Blue color component for Fog and distance..

[BACKGROUND]
That variable need 3parameters lines.These lines must contain in order :
Red , Green , Blue color component for background color..

[TEXTUREMEM]
Will define if 3DMapEditor will use graphic card memory ( VIDEORAM ) for textures or.
system memory ( RAMONLY )

[TEXTURETYPE]
If you choose to use VIDEORAM then you also can use Texture compression if available ( TRUE ) or no compression ( FALSE )

[MATRIXSIZE]
Will define the size of ghost matrix for positionning..

[MAPPATH]
Define Relative path for maps access

[TEXTUREPATH]
Define Relative path for textures (.jpg , .bmp) access

[3DOBJECTPATH]
Define Relative path for 3dobjects (.x , .3ds) access

[3DSOUNDPATH]
Define Relative path for 3dsounds (.wav , .mp3) access

[DEBUGMODE]
Activate Debug-Mode . Do not use this ! ( Only for debugging )
