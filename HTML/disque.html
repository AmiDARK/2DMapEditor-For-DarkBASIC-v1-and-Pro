3D Map Editor Ver 2.00+

( Html documentation )

Icones DISQUE actifs dans la version 2.11 :

" LOAD MAP " : permet de charger une carte crée avec 3DMap Edit.

" SAVE MAP " : permet de sauvegarder la carte en cours.

" CLEAR MAP " : Efface tout les composants de la carte en cours. ( objets , textures , lumières )

" QUIT " : Quitte l'éditeur pour revenir sur windows.

" LOAD TEXTURE " : Permet d'ajouter une texture sur disque dans la carte en cours.
